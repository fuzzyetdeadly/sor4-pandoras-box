# Nightly builds

Nightly builds are potentially unstable builds of the tool, containing features that are still works in progress. 

That being said, please remember to backup your work, in case there are bugs that might compromise your work. I strongly recommending how to use Git version control to backup your work if you're able to figure it out, as it can make it a lot easier to track changes to mods you're working on.

They are also used for open beta testing, to get early community feedback to help make the tool better.
